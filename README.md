# nodejs-package-practice
使用node.js操作檔案，包含讀取、編輯、轉換...等等。
## 讀取&操作檔案
使用`fs-extra`，完整繼承了之前`fs`。
```bash
npm install fs-extra
```
使用lodash進行資料的處理。
```bash
npm i --save lodash
```
